该项目为Visual Studio Code的数据库客户端扩展, 意在让你的生命更加简单, 支持**MySQL/MariaDB, PostgreSQL, SQLite, Redis**以及**ElasticSearch**的管理, 且可以作为一个SSH客户端.

## 功能说明

为了让软件能够保持维护, 最新版本已经闭源, 你可以免费使用该软件, 但无法使用全部功能, 中国区高级版价格为68元1年.

免费版和付费高级版的功能比较:

| 功能点                                           | 免费版 | 付费版 |
| ------------------------------------------------ | ------ | ------ |
| 连接至MySQL, PostgreSQL, Redis等多种类型的数据库 | ✔     | ✔     |
| 作为SSH客户端连接SSH服务器                       | ✔     | ✔     |
| 对表数据和数据库对象进行增删改查                 | ✔     | ✔     |
| SQL编写补全                                      | ✔     | ✔     |
| 数据库备份恢复                                   | ✔     | ✔     |
| 设置云同步                                       | ✘     | ✔     |
| 更加智能的SQL编写补全                            | ✘     | ✔     |
| 工作空间级别的连接                               | ✘     | ✔     |
| SFTP支持                                         | ✘     | ✔     |
| 将表数据导出为JSON, CSV, XLSX格式                | ✘     | ✔     |
| 打开数据库的操作控制台                           | ✘     | ✔     |
| 查看表的建表语句                                 | ✘     | ✔     |
| 查看SQL执行历史记录                              | ✘     | ✔     |
